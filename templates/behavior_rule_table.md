# 行为规则表模板

## 使用说明

用"当 [触发条件] → NPC [反应] → 玩家 [影响]"的格式编写 2-4 条行为规则。

---

## [NPC 名称] 行为规则表

| # | 触发条件 (Trigger) | NPC 反应 (Response) | 玩家影响 (Player Impact) |
|---|-------------------|-------------------|-----------------------|
| 1 | | | |
| 2 | | | |
| 3 | | | |
| 4 | | | |

---

## 填写指南

### 触发条件 (Trigger) 的类型
- **状态触发**：玩家达到某个状态时（HP<30%、持有特定道具、等级达到 X）
- **行为触发**：玩家执行某个操作时（快速点击、长时间不动、反复刷同一区域）
- **时间触发**：特定时间点（每天早晨、第 N 天、完成 X 次后）
- **位置触发**：玩家进入/离开某个区域
- **进度触发**：完成/未完成某个任务

### NPC 反应 (Response) 的类型
- **对话**：说一句话/一段话
- **行为**：做一个动作（泡茶、走开、给你东西）
- **状态变化**：情绪变化、位置移动、外观变化
- **规则改变**：改变游戏参数（降低操作速度、解锁新选项）

### 玩家影响 (Player Impact) 的类型
- **信息获得**：知道了之前不知道的事
- **情绪变化**：被安慰、被挑战、被吓到
- **行为改变**：换个方式玩、放慢速度、注意某个东西
- **资源变化**：获得/失去物品

## 示例：温柔老奶奶（治愈经营）

| # | 触发条件 | NPC 反应 | 玩家影响 |
|---|---------|---------|---------|
| 1 | 玩家连续快速点击 5 次以上 | 提醒"茶要慢慢泡呢"，画面出现舒缓动画 | 降低操作焦虑，放慢游戏节奏 |
| 2 | 玩家连续失败 2 次泡茶小游戏 | 触发安慰对话，赠送一个"简化泡茶道具" | 降低挫败感，获得辅助工具 |
| 3 | 玩家完成茶馆升级 | 解锁一段老奶奶的回忆故事 + 隐藏配方 | 获得叙事奖励和新内容 |
| 4 | 玩家连续 3 天未访问老奶奶 | 下次对话时表达想念，但不过度催促 | 感受到 NPC 的关心，但无压力 |

## 示例：可疑商人（RPG）

| # | 触发条件 | NPC 反应 | 玩家影响 |
|---|---------|---------|---------|
| 1 | 玩家首次购买物品 | 以"特别优惠价"出售，但物品有隐藏副作用 | 引入风险-奖励决策 |
| 2 | 玩家连续购买 3 次 | 解锁"会员"隐藏商品列表，价格更高但更强力 | 产生信任-怀疑的张力 |
| 3 | 玩家拒绝购买 | 降价 20% 但附带一句可疑的补充说明 | 制造信息不对称的决策压力 |
