# 用户输入示例：废土伙伴女孩

## 用户的初始输入

> 我想做一个在废土世界里遇到的小女孩 NPC，她会跟着玩家走一段路，可以帮玩家发现隐藏物品和提醒危险。
> 她不太说话，但会用动作和表情表达，玩家慢慢会和她建立感情。
> 游戏类型是末日生存探索类的。

## Skill 识别的关键信息

| 维度 | 提取内容 |
|-----|---------|
| NPC 类型 | 小女孩，话少，用动作表达 |
| 游戏类型 | 末日生存探索 |
| 核心功能 | 发现隐藏物品、提醒危险、跟随 |
| 用户期望 | 建立感情、不是纯工具人 |
| 适合方向 | T09 Companion NPC + T08 Silent Guide NPC |

## 预期 Skill 介入策略

1. Step 1: 确定"生存+探索"的体验目标（紧张/发现/陪伴之间的节奏）
2. Step 2-3: 建立角色概念和接触点
3. Template Checkpoint: 推荐 T09 (Companion) + T08 (Silent Guide) 组合
4. Step 5: 特别关注 Player Burden 和 Agency——不要让伙伴成为负担
5. Step 9: 与 C002 Ellie 和 C007 Trico 进行案例比对
